# <a name="labor-cost-tracking-on-projects"></a><span data-ttu-id="0f2aa-103">Kostnadssporing av arbeid på prosjekter</span><span class="sxs-lookup"><span data-stu-id="0f2aa-103">Labor cost tracking on projects</span></span>

<span data-ttu-id="0f2aa-104">_**Gjelder for:** Project Operations for ressursbaserte/ikke-lagerbaserte scenarioer, Lite-distribusjon – avtale til proformafakturering_</span><span class="sxs-lookup"><span data-stu-id="0f2aa-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="0f2aa-105">Dynamics 365 Project Operations sporer arbeidsestimater og forbruk til lavest mulig detaljstyring i prosjektplanen.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-105">Dynamics 365 Project Operations tracks labor estimates and spend at the lowest required granularity on the project plan.</span></span> <span data-ttu-id="0f2aa-106">Det økonomiske estimatet for arbeidskostnader er basert på planlagt innsats og de generiske eller navngitte ressursene som er tilordnet hver bladnodeoppgave i prosjektplanen.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-106">The financial estimate of labor cost is based on the planned effort and the generic or named resources that are assigned to each leaf-node task in the project plan.</span></span> <span data-ttu-id="0f2aa-107">Når prosjektet starter og personene begynner å rapportere tid på ulike prosjektoppgaver, oppsummeres det faktiske forbruket av arbeid som starter en beregning av prognosene.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-107">When the project begins and people start to report time on various project tasks, the actual spend on labor is summarized which starts a calculation of the projections.</span></span>

## <a name="labor-cost-tracking-view"></a><span data-ttu-id="0f2aa-108">Visningen Kostnadssporing av arbeid</span><span class="sxs-lookup"><span data-stu-id="0f2aa-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="0f2aa-109">På **Prosjekter**-siden, på **Sporing**-fanen, kan du velge **Sporing** > **Kostnad** for å åpne **Kostnadssporing**-visningen og se fremdriften av arbeid brukt på hver oppgave i prosjektplanen.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="0f2aa-110">Denne visningen sammenligner også de faktiske arbeidskostnadene som er brukt på en aktivitet, med aktivitetens planlagte arbeidskostnader.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="0f2aa-111">Project Operations bruker følgende formler til å beregne måledata for arbeidskostnader:</span><span class="sxs-lookup"><span data-stu-id="0f2aa-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="0f2aa-112">**Planlagt kostnad**: Beregnet kostnad for alle ressurstilordninger for hver oppgave i bladnode</span><span class="sxs-lookup"><span data-stu-id="0f2aa-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>
- <span data-ttu-id="0f2aa-113">**Faktisk kostnad**: Summen av alle faktiske kostnader for tid som er registrert for oppgaven</span><span class="sxs-lookup"><span data-stu-id="0f2aa-113">**Actual Cost**: Sum of all cost actuals for time recorded on the task</span></span>
- <span data-ttu-id="0f2aa-114">**Kostnadsforbruk i prosent**: Faktisk kostnad ÷ Kostnadsestimat ved fullføring</span><span class="sxs-lookup"><span data-stu-id="0f2aa-114">**Cost consumption percentage**: Actual cost ÷ Cost estimate at complete</span></span>
- <span data-ttu-id="0f2aa-115">**Gjenstående kostnad**: Kostnadsestimat ved fullføring – Faktisk kostnad</span><span class="sxs-lookup"><span data-stu-id="0f2aa-115">**Remaining Cost**: Cost estimate at complete  – Actual cost</span></span>
- <span data-ttu-id="0f2aa-116">**Kostnad ved fullføring**: Gjenstående kostnad + Faktisk kostnad</span><span class="sxs-lookup"><span data-stu-id="0f2aa-116">**Cost at Complete**: Remaining cost + Actual cost</span></span>
- <span data-ttu-id="0f2aa-117">**Kostnadsavvik**: Planlagt kostnad – Kostnadsestimat ved fullføring</span><span class="sxs-lookup"><span data-stu-id="0f2aa-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>

<span data-ttu-id="0f2aa-118">Hver oppgave viser en projisering av kostnadsavviket for oppgaven.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="0f2aa-119">Hvis kostnadsestimatet ved fullføring er mer enn planlagt kostnad, beregnes det at aktiviteten skal gå over budsjettet.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="0f2aa-120">Hvis kostnadsestimatet ved fullføring er mindre enn planlagt kostnad, beregnes det at aktiviteten skal fullføres under budsjettet.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

>[!NOTE]
> <span data-ttu-id="0f2aa-121">Project Operations viser bare arbeidskostnader på **Prosjekt**-siden på **Sporing**-fanen. This tab is designed to work only for reprojecting labor costs by reprojecting effort.</span></span>
<span data-ttu-id="0f2aa-122">Alle kostnadsbeløp som vises, konverteres til prosjektets kostnadsvaluta fra prosjektets kostprisvaluta som brukes til å fastsette kostnadssatsen.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-122">All cost amounts shown are converted to the project's cost currency from the project's cost price currency used to determine the cost rate.</span></span> <span data-ttu-id="0f2aa-123">Kostnadsvalutaen for prosjektet er valutaen til kontraktsenheten i prosjektet.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-123">The cost currency of the project is the currency of the contracting unit on the project.</span></span> <span data-ttu-id="0f2aa-124">Det kan hende at de estimerte kostnadsverdiene for tiden som vises på **Estimater**-fanen på **Prosjekt**-siden, ikke utgjør hele den planlagte kostnadsen på **Sporing**-fanen. Årsaken til dette avviket er forskjellene i hvordan estimert kostnad opppsummeres i rutenettet **Estimater**, og hvordan planlagt kostnad beregnes i rutenettet **Sporing**.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-124">The estimated cost values for time that are shown on the **Estimates** tab on the **Project** page might not add up to the planned cost on the **Tracking** tab. The reason for this discrepancy is because of the differences in how estimated cost is summarized on the **Estimates** grid and how planned cost is calculated on the **Tracking** grid.</span></span> 
>
> - <span data-ttu-id="0f2aa-125">**Estimater**-fanen beregner den estimerte kostnaden ved å bruke samme valuta for kostnadssatsen i prislisten.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-125">The **Estimates tab** calculates the estimated cost by using the same currency of the cost rate in the price list.</span></span> <span data-ttu-id="0f2aa-126">Deretter konverteres den beregnede kostnaden i prislistevalutaen til den beregnede kostnaden i prosjektets kostnadsvaluta.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-126">Then, the estimated cost in the price list currency is converted to the estimated cost in the project's cost currency.</span></span> <span data-ttu-id="0f2aa-127">Beregnet kostnad i prosjektvalutaen vises avrundet til to desimaler.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-127">The estimated cost in the project currency is shown rounded to 2 decimal places.</span></span> <span data-ttu-id="0f2aa-128">Valutapresisjon brukes ikke under konverteringen.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-128">At no point during this conversion is currency precision applied.</span></span> 
> - <span data-ttu-id="0f2aa-129">På **Sporing**-fanen følger beregningen av planlagte kostnader en litt annen beregningsrekkefølge som innebærer at valutapresisjon brukes i to faser:</span><span class="sxs-lookup"><span data-stu-id="0f2aa-129">On the **Tracking** tab, the planned cost calculation follows a slightly different calculation order that involves currency precision being applied in two stages:</span></span> 
   ><ol>
   ><li><span data-ttu-id="0f2aa-130">Det beregnede kostnadsbeløpet i prislistevalutaen konverteres til standardvalutaen (konvertering 1).</span><span class="sxs-lookup"><span data-stu-id="0f2aa-130">The estimated cost amount in the price list currency is converted to the base currency (conversion 1).</span></span></li>
   ><li><span data-ttu-id="0f2aa-131">Det beregnede kostnadsbeløpet i basisvalutaen konverteres til prosjektkostnadsvalutaen (konvertering 2).</span><span class="sxs-lookup"><span data-stu-id="0f2aa-131">The estimated cost amount in the base currency is converted to the project cost currency (conversion 2).</span></span> </li>
   ></ol>
   ><span data-ttu-id="0f2aa-132">Valutapresisjon brukes i begge trinnene for å få en planlagt kostnad (på **Sporing**-fanen) som avviker noe fra den estimerte kostnaden (i visningen **Tidsinndelt** på **Estimater**-fanen).</span><span class="sxs-lookup"><span data-stu-id="0f2aa-132">Currency precision is applied in both steps to get a planned cost (on the **Tracking** tab) that deviates slightly from the estimated cost (on the **Time - phased** view on the **Estimates** tab).</span></span> 
   
## <a name="reprojecting-costs-on-leaf-node-tasks"></a><span data-ttu-id="0f2aa-133">Projisere kostnader på nytt for bladnodeoppgaver</span><span class="sxs-lookup"><span data-stu-id="0f2aa-133">Reprojecting costs on leaf node tasks</span></span>

<span data-ttu-id="0f2aa-134">Arbeidskostnader i en bladnodeoppgave kan ikke løses på nytt direkte i kategorien **Sporing** på **Prosjekt**-siden.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-134">Labor costs on a leaf node task can't be directly reprojected on the **Tracking** tab on the **Project page**.</span></span> <span data-ttu-id="0f2aa-135">Du kan imidlertid bruke visningen **Innsatssporing** til å utføre gjenstående innsats på en oppgave på nytt.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-135">However, you can use the **Effort Tracking** view to reproject the remaining effort on a task.</span></span> <span data-ttu-id="0f2aa-136">Dette fører til en omberegning av de gjenstående kostnadene for aktiviteten.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-136">This causes a recalculation of the remaining cost on the task.</span></span> <span data-ttu-id="0f2aa-137">Nedenfor finner du en beskrivelse av hvordan dette fungerer.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-137">The following is a description of how this works.</span></span>

1. <span data-ttu-id="0f2aa-138">En prosjektleder kan projisere innsats på oppgaver på nytt ved å oppdatere standardverdien i **Gjenstående innsats**-feltet med et nytt overslag over gjenstående innsats for aktiviteten.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-138">A project manager can reproject effort on tasks by updating the default value in the **Remaining Effort** field with a new estimate of the remaining effort on the task.</span></span> <span data-ttu-id="0f2aa-139">Ny projisering fører til en omberegning av oppgavens innsatsestimat ved fullføring, fremdriftsprosenten og det beregnede innsatsavviket for en oppgave.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-139">Reprojecting causes a recalculation of the task's effort estimate at complete (EAC), progress percentage, and the projected effort variance on a task.</span></span> <span data-ttu-id="0f2aa-140">EAC, estimatet for fullføring (ETC) og fremdriftsprosenten på aktivitetssammendragene beregnes også på nytt, og det lages en ny projisering av innsatsavvik.</span><span class="sxs-lookup"><span data-stu-id="0f2aa-140">The EAC, estimate to complete (ETC), and progress percentage on the summary tasks are also recalculated and produce a new projection of effort variance.</span></span>
