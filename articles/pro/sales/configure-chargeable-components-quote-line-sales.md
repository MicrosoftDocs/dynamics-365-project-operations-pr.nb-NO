# <a name="configure-the-chargeable-components-of-a-quote-line"></a><span data-ttu-id="52466-103">Konfigurere de belastbare komponentene i en tilbudslinje</span><span class="sxs-lookup"><span data-stu-id="52466-103">Configure the chargeable components of a quote line</span></span>

<span data-ttu-id="52466-104">_**Gjelder:** Lite-distribusjon – avtale til proformafakturering_</span><span class="sxs-lookup"><span data-stu-id="52466-104">_**Applies To:** Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="52466-105">En prosjektbasert tilbudslinje har konseptet *inkluderte* komponenter og *belastbare* komponenter.</span><span class="sxs-lookup"><span data-stu-id="52466-105">A project-based quote line has the concept of *included* components and *chargeable* components.</span></span>

<span data-ttu-id="52466-106">Inkluderte komponenter er underlagt følgende:</span><span class="sxs-lookup"><span data-stu-id="52466-106">Included components are subject to:</span></span>

  - <span data-ttu-id="52466-108">Må ikke overskrides-grenser</span><span class="sxs-lookup"><span data-stu-id="52466-108">Not-to-exceed limits</span></span> 
  - <span data-ttu-id="52466-109">Innstillinger for fakturafrekvens som er definert på den prosjektbaserte tilbudslinjen</span><span class="sxs-lookup"><span data-stu-id="52466-109">Invoice frequency settings defined on the project-based quote line</span></span>

<span data-ttu-id="52466-110">Et delsett av de inkluderte komponentene kan merkes som belastbarte ved hjelp av feltet **Faktureringstype**.</span><span class="sxs-lookup"><span data-stu-id="52466-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="52466-111">Feltet **Faktureringstype** er et alternativsett som tillater følgende verdier i konteksten for en tilbudslinje:</span><span class="sxs-lookup"><span data-stu-id="52466-111">The **Billing Type** field is an option-set that allows the following values in the context of a quote line:</span></span>

  - <span data-ttu-id="52466-112">Belastbar</span><span class="sxs-lookup"><span data-stu-id="52466-112">Chargeable</span></span>
  - <span data-ttu-id="52466-113">Ikke-belastbar</span><span class="sxs-lookup"><span data-stu-id="52466-113">Non-chargeable</span></span>

<span data-ttu-id="52466-114">Belastbare komponenter kan defineres for oppgaver, roller og transaksjonskategorier.</span><span class="sxs-lookup"><span data-stu-id="52466-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="52466-117">Belastbarhet er definert for roller for en tilbudslinje og gjelder bare transaksjonsklassen **Tid**.</span><span class="sxs-lookup"><span data-stu-id="52466-117">Chargeability is defined on roles for a quote line and only applies to the **Time** transaction class.</span></span> <span data-ttu-id="52466-118">Hvis feltet **Inkluder td** er satt til **Nei** på en prosjekttilbudslinje, er ikke **Belastbare roller** -fanen tilgjengelig.</span><span class="sxs-lookup"><span data-stu-id="52466-118">If the field, **Include Time** is set to **No** on a project quote line, the **Chargeable Roles** tab isn't available.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="52466-121">Oppdatere en prosjektoppgave til å være belastbar eller ikke-belastbar</span><span class="sxs-lookup"><span data-stu-id="52466-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="52466-122">En prosjektoppgave kan være belastbar eller ikke-belastbar i konteksten til en bestemt prosjektbasert tilbudslinje, noe som gjør følgende oppsett mulig:</span><span class="sxs-lookup"><span data-stu-id="52466-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>
