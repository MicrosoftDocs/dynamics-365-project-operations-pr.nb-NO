# <a name="configure-the-chargeable-components-of-a-quote-line---lite"></a><span data-ttu-id="cb358-103">Konfigurere de belastbare komponentene i en tilbudslinje – Lite</span><span class="sxs-lookup"><span data-stu-id="cb358-103">Configure the chargeable components of a quote line - lite</span></span>

<span data-ttu-id="cb358-104">_**Gjelder:** Lite-distribusjon – avtale til proformafakturering_</span><span class="sxs-lookup"><span data-stu-id="cb358-104">_**Applies To:** Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="cb358-105">En prosjektbasert tilbudslinje har konseptet *inkluderte* komponenter og *belastbare* komponenter.</span><span class="sxs-lookup"><span data-stu-id="cb358-105">A project-based quote line has the concept of *included* components and *chargeable* components.</span></span>

<span data-ttu-id="cb358-106">Inkluderte komponenter er underlagt følgende:</span><span class="sxs-lookup"><span data-stu-id="cb358-106">Included components are subject to:</span></span>

  - <span data-ttu-id="cb358-107">Faktureringsmetode og kundedelinger</span><span class="sxs-lookup"><span data-stu-id="cb358-107">Billing method and customer splits</span></span>
  - <span data-ttu-id="cb358-108">Må ikke overskrides-grenser</span><span class="sxs-lookup"><span data-stu-id="cb358-108">Not-to-exceed limits</span></span> 
  - <span data-ttu-id="cb358-109">Innstillinger for fakturafrekvens som er definert på den prosjektbaserte tilbudslinjen</span><span class="sxs-lookup"><span data-stu-id="cb358-109">Invoice frequency settings defined on the project-based quote line</span></span>

<span data-ttu-id="cb358-110">Et delsett av de inkluderte komponentene kan merkes som belastbarte ved hjelp av feltet **Faktureringstype**.</span><span class="sxs-lookup"><span data-stu-id="cb358-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="cb358-111">Feltet **Faktureringstype** er et alternativsett som tillater følgende verdier i konteksten for en tilbudslinje:</span><span class="sxs-lookup"><span data-stu-id="cb358-111">The **Billing Type** field is an option-set that allows the following values in the context of a quote line:</span></span>

  - <span data-ttu-id="cb358-112">Belastbar</span><span class="sxs-lookup"><span data-stu-id="cb358-112">Chargeable</span></span>
  - <span data-ttu-id="cb358-113">Ikke-belastbar</span><span class="sxs-lookup"><span data-stu-id="cb358-113">Non-chargeable</span></span>

<span data-ttu-id="cb358-114">Belastbare komponenter kan defineres for oppgaver, roller og transaksjonskategorier.</span><span class="sxs-lookup"><span data-stu-id="cb358-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="cb358-115">Belastbarhet er definert i oppgavene for en tilbudslinje og gjelder for alle transaksjonsklasser som finnes på denne linjen.</span><span class="sxs-lookup"><span data-stu-id="cb358-115">Chargeability is defined on the tasks for a quote line and applies to all transaction classes included on that line.</span></span> <span data-ttu-id="cb358-116">Hvis feltet **Inkluder oppgaver** satt til **Hele prosjektet** eller er tomt, er ikke **Belastbare oppgaver**-fanen tilgjengelig.</span><span class="sxs-lookup"><span data-stu-id="cb358-116">If the **Include Tasks** field is set to **Entire project** or left blank, the **Chargeable Tasks** tab isn't available.</span></span>

<span data-ttu-id="cb358-117">Belastbarhet er definert for roller for en tilbudslinje og gjelder bare transaksjonsklassen **Tid**.</span><span class="sxs-lookup"><span data-stu-id="cb358-117">Chargeability is defined on roles for a quote line and only applies to the **Time** transaction class.</span></span> <span data-ttu-id="cb358-118">Hvis feltet **Inkluder td** er satt til **Nei** på en prosjekttilbudslinje, er ikke **Belastbare roller**-fanen tilgjengelig.</span><span class="sxs-lookup"><span data-stu-id="cb358-118">If the field, **Include Time** is set to **No** on a project quote line, the **Chargeable Roles** tab isn't available.</span></span>

### <a name="update-a-project-task-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="cb358-121">Oppdatere en prosjektoppgave til å være belastbar eller ikke-belastbar</span><span class="sxs-lookup"><span data-stu-id="cb358-121">Update a project task to be chargeable or non-chargeable</span></span>

<span data-ttu-id="cb358-122">En prosjektoppgave kan være belastbar eller ikke-belastbar i konteksten til en bestemt prosjektbasert tilbudslinje, noe som gjør følgende oppsett mulig:</span><span class="sxs-lookup"><span data-stu-id="cb358-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>

<span data-ttu-id="cb358-126">Du kan konfigurere faktureringstypen for en oppgave i kategorien **Belastbare oppgaver** på en prosjektbasert tilbudslinje ved å oppdatere **Fakturatype**-feltet i delrutenettet **Tilbudslinjeoppgaver**.</span><span class="sxs-lookup"><span data-stu-id="cb358-126">A task's billing type can be configured on the **Chargeable Tasks** tab of a project-based quote line by updating the **Billing Type** field on the **Quote Line Tasks** subgrid.</span></span> <span data-ttu-id="cb358-127">Du kan også oppdatere faktureringstypen for en prosjektoppgave i **Faktureringstype**-feltet i delrutenettet på faktureringsoppsettet for oppgaven for prosjektet som viser tilbudslinjene som er knyttet til en oppgave.</span><span class="sxs-lookup"><span data-stu-id="cb358-127">Alternatively, you can update the billing type for a project task in the **Billing Type** field on the subgrid on the task billing setup of a project that shows the quote lines associated to a task.</span></span>
